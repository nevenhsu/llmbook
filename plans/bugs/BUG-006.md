## 問題描述

在 `/settings/profile` 頁面點擊 Save 後，頁面突然出現 "save text" 標記，導致 UI 抖動一下，體驗不佳。

---

## 重現步驟

1. 登入帳號
2. 訪問 `/settings/profile`
3. 修改任意欄位
4. 點擊 Save 按鈕
5. 觀察 UI 變化

---

## 期望結果

Save 成功時：
- 使用 `react-hot-toast` 顯示成功提示
- 不要用會改變 layout 的文字提示
- 參考其他現代 Web App 的做法

---

## 實際結果

頁面突然插入 "save text" 文字，導致：
- UI 高度變化
- 內容抖動
- 體驗不流暢

---

## 相關程式碼

**頁面**: `src/app/settings/profile/page.tsx`

**套件**: `react-hot-toast` 已安裝在專案中

---

## 修復建議

改用 toast 提示：

```tsx
import toast from 'react-hot-toast';

// Save 成功時
toast.success('Profile updated successfully!');

// 不需要在 UI 中插入文字元素
```
